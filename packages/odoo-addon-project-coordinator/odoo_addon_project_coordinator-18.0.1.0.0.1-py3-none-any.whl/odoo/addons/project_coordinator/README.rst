This module adds an Project Coordinator for projects and displays it in
all tasks of the project.

.. contents::
   :local:

Usage
=====

To use this module, you need to:

1. Go to the projects module.
2. Edit or Create a new project.
3. Select a Project Coordinator.

To view it from the task:

1. Go to the project module -> Tasks
2. Edit or Create a new task
3. You will see that in the Project Coordinator of the project will be
   displayed.
