Узел Replace Text выполняет простую замену текста. Он ищет указанный фрагмент текста во входных данных и заменяет каждое его вхождение на новый фрагмент. Операция применяется ко всем текстовым входам, переданным узлу.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Да | - | Текст для обработки. |
| `find` | STRING | Нет | - | Текст, который нужно найти и заменить (по умолчанию: пустая строка). |
| `replace` | STRING | Нет | - | Текст, на который нужно заменить найденный фрагмент (по умолчанию: пустая строка). |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `text` | STRING | Обработанный текст, в котором все вхождения текста `find` заменены на текст `replace`. |
