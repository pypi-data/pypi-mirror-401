El nodo Text to Uppercase toma una entrada de texto y convierte todos sus caracteres a mayúsculas. Es una utilidad simple de procesamiento de texto que modifica la capitalización de la cadena proporcionada.

## Entradas

| Parámetro | Tipo de Dato | Requerido | Rango | Descripción |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Sí | N/A | La cadena de texto que se convertirá a mayúsculas. |

## Salidas

| Nombre de Salida | Tipo de Dato | Descripción |
|-------------|-----------|-------------|
| `text` | STRING | El texto resultante con todos los caracteres convertidos a mayúsculas. |
