O nó Text to Uppercase recebe um texto de entrada e converte todos os seus caracteres para letras maiúsculas. É uma utilidade simples de processamento de texto que modifica a caixa da string fornecida.

## Entradas

| Parâmetro | Tipo de Dados | Obrigatório | Intervalo | Descrição |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Sim | N/A | A string de texto a ser convertida para letras maiúsculas. |

## Saídas

| Nome da Saída | Tipo de Dados | Descrição |
|-------------|-----------|-------------|
| `text` | STRING | O texto resultante, com todos os caracteres convertidos para letras maiúsculas. |
