**Metni Büyük Harfe Dönüştür** düğümü, bir metin girdisi alır ve tüm karakterlerini büyük harfe dönüştürür. Sağlanan dizenin büyük/küçük harf durumunu değiştiren basit bir metin işleme yardımcısıdır.

## Girdiler

| Parametre | Veri Türü | Zorunlu | Aralık | Açıklama |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Evet | Yok | Büyük harfe dönüştürülecek metin dizesi. |

## Çıktılar

| Çıktı Adı | Veri Türü | Açıklama |
|-------------|-----------|-------------|
| `text` | STRING | Tüm karakterleri büyük harfe dönüştürülmüş sonuç metni. |
