Nome do Nó: Reroute Node
Propósito do Nó: Usado principalmente para organizar a lógica de linhas de conexão excessivamente longas no fluxo de trabalho do ComfyUI.

## Como Usar os Nós Reroute

| Opções do Menu | Descrição |
| --- | --- |
| Show Type | Exibe a propriedade de tipo do nó |
| Hide Type By Default | Oculta a propriedade de tipo do nó por padrão |
| Set Vertical | Define a direção da fiação do nó como vertical |
| Set Horizontal | Define a direção da fiação do nó como horizontal |

Quando a lógica de fiação está muito longa e complexa, e você deseja organizar a interface, pode inserir um nó ```Reroute``` entre dois pontos de conexão. A entrada e a saída deste nó não possuem restrição de tipo, e o estilo padrão é horizontal. Você pode alterar a direção da fiação para vertical através do menu de clique com o botão direito.
