Узел LTXV Audio VAE Decode преобразует латентное представление аудио обратно в аудиоволну. Он использует специализированную модель Audio VAE для выполнения этого процесса декодирования, создавая аудиовыход с определённой частотой дискретизации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | Да | N/A | Латентное представление для декодирования. |
| `audio_vae` | VAE | Да | N/A | Модель Audio VAE, используемая для декодирования латентного представления. |

**Примечание:** Если предоставленное латентное представление вложенное (содержит несколько латентов), узел автоматически использует последний латент в последовательности для декодирования.

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `Audio` | AUDIO | Декодированная аудиоволна и связанная с ней частота дискретизации. |
