## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الإطار الأول الذي سيُستخدم في الفيديو. |
| `model` | COMBO | نعم | `"LTX-2 (Fast)"`<br>`"LTX-2 (Quality)"` | نموذج الذكاء الاصطناعي المستخدم في توليد الفيديو. النموذج "Fast" مُحسَّن للسرعة، بينما يركز نموذج "Quality" على الدقة البصرية. |
| `prompt` | STRING | نعم | - | وصف نصي يوجه محتوى وحركة الفيديو المُنشأ. |
| `duration` | COMBO | نعم | `6`<br>`8`<br>`10`<br>`12`<br>`14`<br>`16`<br>`18`<br>`20` | طول الفيديو بالثواني (القيمة الافتراضية: 8). |
| `resolution` | COMBO | نعم | `"1920x1080"`<br>`"2560x1440"`<br>`"3840x2160"` | دقة الإخراج للفيديو المُنشأ. |
| `fps` | COMBO | نعم | `25`<br>`50` | عدد الإطارات في الثانية للفيديو (القيمة الافتراضية: 25). |
| `generate_audio` | BOOLEAN | لا | - | عند تفعيله (True)، سيتضمن الفيديو المُنشأ صوتًا مُولَّدًا بالذكاء الاصطناعي يتناسب مع المشهد (القيمة الافتراضية: False). |

**قيود هامة:**
*   يجب أن يحتوي مدخل `image` على صورة واحدة بالضبط.
*   يجب أن يكون طول `prompt` بين 1 و 10,000 حرف.
*   إذا اخترت `duration` أطول من 10 ثوانٍ، فيجب عليك استخدام نموذج **"LTX-2 (Fast)"** ودقة **"1920x1080"** و **25** إطارًا في الثانية. هذا المزيج مطلوب لمقاطع الفيديو الطويلة.
