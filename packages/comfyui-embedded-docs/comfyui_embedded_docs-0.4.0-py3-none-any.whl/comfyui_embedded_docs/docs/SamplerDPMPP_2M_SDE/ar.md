```markdown
عُقدة SamplerDPMPP_2M_SDE تُنشئ عيِّنًا من نوع DPM++ 2M SDE لنماذج الانتشار. يستخدم هذا العيِّن حلّالات المعادلات التفاضلية من الرتبة الثانية مع المعادلات التفاضلية العشوائية لتوليد العينات. يوفر أنواعًا مختلفة من الحلّالات وخيارات معالجة الضوضاء للتحكم في عملية أخذ العينات.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `نوع المُحلل` | STRING | نعم | `"midpoint"`<br>`"heun"` | نوع حلّ المعادلات التفاضلية المستخدم في عملية أخذ العينات |
| `إيتا` | FLOAT | نعم | 0.0 - 100.0 | يتحكم في العشوائية أثناء عملية أخذ العينات (القيمة الافتراضية: 1.0) |
| `ضجيج s` | FLOAT | نعم | 0.0 - 100.0 | يتحكم في مقدار الضوضاء المضافة أثناء أخذ العينات (القيمة الافتراضية: 1.0) |
| `جهاز الضجيج` | STRING | نعم | `"gpu"`<br>`"cpu"` | الجهاز الذي تُجرى عليه حسابات الضوضاء |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `sampler` | SAMPLER | كائن عيِّن مُهيأ وجاهز للاستخدام في خطورة أخذ العينات |
```
