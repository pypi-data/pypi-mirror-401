تأخذ عقدة تحويل النص إلى أحرف صغيرة سلسلة نصية كمدخل وتحول جميع أحرفها إلى أحرف صغيرة. إنها أداة بسيطة لتوحيد حالة النص.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | نعم | أي سلسلة نصية | السلسلة النصية المراد تحويلها إلى أحرف صغيرة. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `text` | STRING | النص المدخل بعد تحويل جميع أحرفه إلى أحرف صغيرة. |
