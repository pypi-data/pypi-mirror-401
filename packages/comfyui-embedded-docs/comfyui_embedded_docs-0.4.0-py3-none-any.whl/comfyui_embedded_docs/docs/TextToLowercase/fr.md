Le nœud Text to Lowercase prend une chaîne de texte en entrée et convertit tous ses caractères en minuscules. C'est un utilitaire simple pour standardiser la casse du texte.

## Entrées

| Paramètre | Type de données | Requis | Plage | Description |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Oui | Toute chaîne de texte | La chaîne de texte à convertir en minuscules. |

## Sorties

| Nom de la sortie | Type de données | Description |
|-------------|-----------|-------------|
| `text` | STRING | Le texte d'entrée avec tous les caractères convertis en minuscules. |
