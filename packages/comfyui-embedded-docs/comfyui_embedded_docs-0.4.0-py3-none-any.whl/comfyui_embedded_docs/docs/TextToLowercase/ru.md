Узел Text to Lowercase принимает текстовую строку на вход и преобразует все её символы в нижний регистр. Это простой инструмент для стандартизации регистра текста.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | Да | Любая текстовая строка | Текстовая строка для преобразования в нижний регистр. |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `text` | STRING | Входной текст со всеми символами, преобразованными в нижний регистр. |
