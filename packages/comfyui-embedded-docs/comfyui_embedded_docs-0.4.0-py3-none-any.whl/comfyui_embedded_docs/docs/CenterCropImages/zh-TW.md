Center Crop Images 節點會從圖像中心裁剪出指定寬度和高度的區域。它會計算輸入圖像的中心區域，並擷取出定義尺寸的矩形區域。如果請求的裁剪尺寸大於原圖像，裁剪範圍將被限制在圖像邊界內。

## 輸入參數

| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | 是 | - | 要進行裁剪的輸入圖像。 |
| `width` | INT | 否 | 1 至 8192 | 裁剪區域的寬度（預設值：512）。 |
| `height` | INT | 否 | 1 至 8192 | 裁剪區域的高度（預設值：512）。 |

## 輸出結果

| 輸出名稱 | 資料類型 | 描述 |
|-------------|-----------|-------------|
| `image` | IMAGE | 經過中心裁剪操作後產生的圖像。 |
