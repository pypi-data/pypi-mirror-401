Set Hook Keyframes 節點允許您對現有的掛鉤群組應用關鍵影格排程功能。它接收一個掛鉤群組，並可選擇性地應用關鍵影格時間資訊，以控制在生成過程中不同掛鉤的執行時機。當提供關鍵影格時，此節點會複製掛鉤群組，並為群組內的所有掛鉤設定關鍵影格時間。

| `hook_kf` | HOOK_KEYFRAMES | 否 | - | 可選的關鍵影格群組，包含掛鉤執行的時間資訊 |
