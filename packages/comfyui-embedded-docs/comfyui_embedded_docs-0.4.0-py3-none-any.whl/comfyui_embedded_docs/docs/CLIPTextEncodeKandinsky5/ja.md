| パラメータ | データ型 | 必須 | 範囲 | 説明 |
|-----------|-----------|----------|-------|-------------|
| `clip` | CLIP | はい | | テキストプロンプトをトークン化およびエンコードするために使用されるCLIPモデルです。 |
| `clip_l` | STRING | はい | | 主要なテキストプロンプトです。この入力は複数行のテキストと動的プロンプトをサポートしています。 |
| `qwen25_7b` | STRING | はい | | 2番目のテキストプロンプトです。この入力は複数行のテキストと動的プロンプトをサポートしています。 |
