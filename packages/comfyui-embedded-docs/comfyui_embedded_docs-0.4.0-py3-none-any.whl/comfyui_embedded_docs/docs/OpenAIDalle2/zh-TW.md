## 運作原理

此節點連接到 OpenAI 的 DALL·E 2 API，根據文字描述建立影像。當您提供文字提示時，節點會將其發送到 OpenAI 的伺服器，伺服器會生成相應的影像並將其傳回 ComfyUI。該節點可以在兩種模式下運作：僅使用文字提示的標準影像生成模式，或在同時提供影像和遮罩時的影像編輯模式。在編輯模式下，它使用遮罩來決定原始影像的哪些部分應該被修改，同時保持其他區域不變。
