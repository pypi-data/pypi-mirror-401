## 概述

Differential Diffusion 節點透過基於時間步長閾值應用二值遮罩來修改去噪過程。它創建一個在原始去噪遮罩和基於閾值的二值遮罩之間混合的遮罩，允許對擴散過程強度進行受控調整。
