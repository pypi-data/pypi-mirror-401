# LTXVConcatAVLatent

تُجمع عقدة LTXVConcatAVLatent التمثيل الكامن للفيديو والتمثيل الكامن للصوت في مخرج كامن واحد مُدمَج. تقوم بدمج موترات `samples` من كلا المدخلين، وإذا كانت موجودة، موترات `noise_mask` الخاصة بهما أيضًا، مما يُهيئها لمزيد من المعالجة في مسار توليد الفيديو.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `video_latent` | LATENT | نعم | | التمثيل الكامن لبيانات الفيديو. |
| `audio_latent` | LATENT | نعم | | التمثيل الكامن لبيانات الصوت. |

**ملاحظة:** يتم دمج موترات `samples` من مدخلي `video_latent` و `audio_latent`. إذا احتوى أي من المدخلين على `noise_mask`، فسيتم استخدامه؛ وإذا كان أحدهما مفقودًا، يتم إنشاء قناع من القيم واحد (بنفس شكل موتر `samples` المقابل) له. ثم يتم أيضًا دمج الأقنعة الناتجة.

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `latent` | LATENT | قاموس كامن واحد يحتوي على موتر `samples` المُدمَج، وإذا كان مُطبقًا، قناع `noise_mask` المُدمَج من مدخلي الفيديو والصوت. |
