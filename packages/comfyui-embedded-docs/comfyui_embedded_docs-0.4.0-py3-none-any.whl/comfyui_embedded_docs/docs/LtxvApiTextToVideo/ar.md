## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `model` | COMBO | نعم | `"LTX-2 (Fast)"`<br>`"LTX-2 (Quality)"`<br>`"LTX-2 (Turbo)"` | نموذج الذكاء الاصطناعي المستخدم في إنشاء الفيديو. النماذج المتاحة معيّنة من `MODELS_MAP` في الكود المصدري. |
| `prompt` | STRING | نعم | - | الوصف النصي الذي سيستخدمه الذكاء الاصطناعي لإنشاء الفيديو. يدخل هذا الحقل عدة أسطر من النص. |
| `duration` | COMBO | نعم | `6`<br>`8`<br>`10`<br>`12`<br>`14`<br>`16`<br>`18`<br>`20` | طول الفيديو المُنشأ بالثواني (القيمة الافتراضية: 8). |
| `resolution` | COMBO | نعم | `"1920x1080"`<br>`"2560x1440"`<br>`"3840x2160"` | أبعاد البكسل (العرض × الارتفاع) للفيديو الناتج. |
| `fps` | COMBO | نعم | `25`<br>`50` | عدد الإطارات في الثانية للفيديو (القيمة الافتراضية: 25). |
| `generate_audio` | BOOLEAN | لا | - | عند التمكين، سيتضمن الفيديو المُنشأ صوتًا مُولّدًا بالذكاء الاصطناعي يتناسب مع المشهد (القيمة الافتراضية: False). |

**قيود مهمة:**
*   يجب أن يكون طول النص في `prompt` بين 1 و 10,000 حرف.
*   إذا اخترت `duration` أكبر من 10 ثوانٍ، فيجب عليك أيضًا استخدام نموذج `"LTX-2 (Fast)"` ودقة `"1920x1080"` و `fps` بقيمة `25`. هذا المزيج مطلوب لمقاطع الفيديو الأطول.
