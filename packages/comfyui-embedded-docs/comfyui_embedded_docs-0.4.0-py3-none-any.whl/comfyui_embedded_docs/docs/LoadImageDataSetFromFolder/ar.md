يُحمّل هذا العقد صورًا متعددة من مجلد فرعي محدد ضمن دليل الإدخال الخاص بـ ComfyUI. حيث يفحص المجلد المختار بحثًا عن أنواع ملفات الصور الشائعة ويعيدها كقائمة، مما يجعله مفيدًا للمعالجة الدفعية أو إعداد مجموعات البيانات.

## المدخلات

| المعامل | نوع البيانات | إلزامي | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `folder` | STRING | نعم | *خيارات متعددة متاحة* | المجلد الذي سيتم تحميل الصور منه. الخيارات هي المجلدات الفرعية الموجودة في دليل الإدخال الرئيسي لـ ComfyUI. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `images` | IMAGE | قائمة بالصور المحملة. يحمل العقد جميع ملفات الصور الصالحة (PNG, JPG, JPEG, WEBP) الموجودة في المجلد المحدد. |
