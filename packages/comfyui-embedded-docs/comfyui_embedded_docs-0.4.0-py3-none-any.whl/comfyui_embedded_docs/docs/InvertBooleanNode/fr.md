Ce nœud prend une seule entrée booléenne (vrai/faux) et renvoie la valeur opposée. Il effectue une opération logique NON, transformant `true` en `false` et `false` en `true`.

## Entrées

| Paramètre | Type de données | Requis | Plage | Description |
|-----------|-----------|----------|-------|-------------|
| `boolean` | BOOLEAN | Oui | `true`<br>`false` | La valeur booléenne d'entrée à inverser. |

## Sorties

| Nom de la sortie | Type de données | Description |
|-------------|-----------|-------------|
| `output` | BOOLEAN | La valeur booléenne inversée. |
