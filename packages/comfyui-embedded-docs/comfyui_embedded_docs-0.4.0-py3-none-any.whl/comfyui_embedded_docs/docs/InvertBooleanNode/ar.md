يأخذ هذا العقدة قيمة منطقية (صواب/خطأ) واحدة كمدخل ويخرج القيمة المعاكسة. يقوم بإجراء عملية منطقية NOT، حيث يحول `true` إلى `false` و `false` إلى `true`.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `boolean` | BOOLEAN | نعم | `true`<br>`false` | القيمة المنطقية المدخلة المراد عكسها. |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `output` | BOOLEAN | القيمة المنطقية المعكوسة. |
