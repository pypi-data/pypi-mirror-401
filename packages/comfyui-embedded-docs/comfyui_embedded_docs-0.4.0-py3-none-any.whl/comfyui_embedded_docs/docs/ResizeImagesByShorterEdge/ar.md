تغيير حجم الصور حسب الحافة الأقصر

تغيير حجم الصور عن طريق ضبط أبعادها بحيث يتطابق طول الحافة الأقصر مع قيمة مستهدفة محددة. يحسب الأبعاد الجديدة للحفاظ على نسبة العرض إلى الارتفاع الأصلية للصورة. يتم إرجاع الصورة بعد تغيير حجمها.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد تغيير حجمها. |
| `shorter_edge` | INT | لا | 1 إلى 8192 | الطول المستهدف للحافة الأقصر. (القيمة الافتراضية: 512) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `image` | IMAGE | الصورة بعد تغيير حجمها. |
