تعدِّل عقدة Adjust Brightness سطوع الصورة المدخلة. تعمل عن طريق ضرب قيمة كل بكسل بعامل محدد، ثم تضمن بقاء القيم الناتجة ضمن نطاق صالح. عامل بقيمة 1.0 يترك الصورة دون تغيير، والقيم الأقل من 1.0 تجعلها أغمق، والقيم الأعلى من 1.0 تجعلها أكثر إشراقًا.

## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `image` | IMAGE | نعم | - | الصورة المدخلة المراد تعديل سطوعها. |
| `factor` | FLOAT | لا | 0.0 - 2.0 | عامل السطوع. 1.0 = لا تغيير، <1.0 = أغمق، >1.0 = أكثر إشراقًا. (القيمة الافتراضية: 1.0) |

## المخرجات

| اسم المخرج | نوع البيانات | الوصف |
|-------------|-----------|-------------|
| `image` | IMAGE | الصورة الناتجة بعد تعديل السطوع. |
