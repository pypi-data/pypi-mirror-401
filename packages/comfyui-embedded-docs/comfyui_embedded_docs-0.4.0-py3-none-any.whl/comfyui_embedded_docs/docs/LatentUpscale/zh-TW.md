LatentUpscale 節點專為放大圖片的潛在表示而設計。它允許調整輸出圖片的尺寸和放大方法，為提升潛在圖像解析度提供了靈活性。

## 輸入參數

| 參數名稱 | 資料類型 | 描述 |
|-----------|-------------|-------------|
| `樣本` | `LATENT`    | 待放大的圖片潛在表示。此參數對於決定放大過程的起始點至關重要。 |
| `放大方法` | COMBO[STRING] | 指定用於放大潛在圖像的方法。不同的方法會影響放大後圖像的品質和特性。 |
| `寬度`   | `INT`       | 放大後圖像的期望寬度。若設為 0，將根據高度計算以維持寬高比。 |
| `高度`  | `INT`       | 放大後圖像的期望高度。若設為 0，將根據寬度計算以維持寬高比。 |
| `裁切`    | COMBO[STRING] | 決定放大後的圖像應如何裁剪，這會影響輸出的最終外觀和尺寸。 |

## 輸出參數

| 參數名稱 | 資料類型 | 描述 |
|-----------|-------------|-------------|
| `latent`  | `LATENT`    | 經過放大的圖片潛在表示，已準備好進行後續處理或生成。 |
