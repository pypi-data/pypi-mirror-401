이 노드는 텍스트 문자열의 시작과 끝에 있는 추가 공백, 탭 또는 줄바꿈을 제거합니다. 텍스트 입력을 받아 앞뒤 공백이 제거된 정리된 버전을 반환합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `text` | STRING | 예 | 해당 없음 | 앞뒤 공백을 제거할 텍스트 문자열입니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `text` | STRING | 모든 앞뒤 공백 문자가 제거된 처리된 텍스트입니다. |
