| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 參數說明 |

| `image` | IMAGE | 是 | - | 用於生成影片的參考圖像 |
| `prompt` | STRING | 是 | - | 影片生成的文字描述（支援多行輸入） |
