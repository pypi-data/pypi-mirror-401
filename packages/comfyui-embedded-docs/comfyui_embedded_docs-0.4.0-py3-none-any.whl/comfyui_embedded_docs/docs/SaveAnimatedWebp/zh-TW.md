此節點專為將一系列影像儲存為動畫 WEBP 檔案而設計。它負責將個別影格匯聚成連貫的動畫，應用指定的元數據，並根據品質和壓縮設定優化輸出。

| 欄位名稱 | 資料類型 | 描述 |
|----------|-----------|------|
| `圖片` | IMAGE | 要儲存為動畫 WEBP 影格的影像列表。此參數對於定義動畫的視覺內容至關重要。 |
| `檔名前綴` | STRING | 指定輸出檔案的基礎名稱，系統會自動附加計數器和 ".webp" 副檔名。此參數對於識別和組織已儲存檔案非常重要。 |

| `無損` | BOOLEAN | 布林值，指示是否使用無損壓縮，這會影響動畫的檔案大小和品質。 |
| `品質` | INT | 介於 0 到 100 之間的值，設定壓縮品質等級，數值越高影像品質越好，但檔案大小也越大。 |
