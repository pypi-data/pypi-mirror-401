يُعدّل عقد Mahiro وظيفة التوجيه لتركيز المزيد على اتجاه الموجه الإيجابي بدلاً من الفرق بين الموجهات الإيجابية والسلبية. يقوم العقد بإنشاء نموذج معدّل يطبق نهجًا مخصصًا لتحجيم التوجيه باستخدام التشابه الجيبي (cosine similarity) بين مخرجات إزالة الضوضاء الشرطية وغير الشرطية المُطَبَّقة. يساعد هذا العقد التجريبي في توجيه عملية التوليد بقوة أكبر نحو الاتجاه المقصود من الموجه الإيجابي.
