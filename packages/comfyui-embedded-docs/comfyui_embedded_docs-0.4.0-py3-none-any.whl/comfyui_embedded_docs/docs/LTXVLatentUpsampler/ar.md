## المدخلات

| المعامل | نوع البيانات | مطلوب | النطاق | الوصف |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | نعم | | التمثيل الكامن (اللاتنت) المُدخل للفيديو المراد تكبير دقته. |
| `upscale_model` | LATENT_UPSCALE_MODEL | نعم | | النموذج المُحمَّل المستخدم لإجراء عملية التكبير بمقدار ضعفين على البيانات الكامنة. |
| `vae` | VAE | نعم | | نموذج VAE المستخدم لإزالة التطبيع من البيانات الكامنة المُدخلة قبل التكبير، ولتطبيع البيانات الكامنة الناتجة بعد ذلك. |
