LTXVImgToVideoInplace 節點透過將輸入圖像編碼到其初始影格中，來對視訊潛在表徵進行條件設定。其工作原理是使用 VAE 將圖像編碼到潛在空間，然後根據指定的強度將其與現有的潛在樣本進行混合。這使得圖像可以作為視訊生成的起點或條件設定訊號。

## 輸入參數

| 參數 | 資料類型 | 必填 | 範圍 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `vae` | VAE | 是 | - | 用於將輸入圖像編碼到潛在空間的 VAE 模型。 |
| `image` | IMAGE | 是 | - | 將被編碼並用於設定視訊潛在表徵條件的輸入圖像。 |
| `latent` | LATENT | 是 | - | 將被修改的目標視訊潛在表徵。 |
| `strength` | FLOAT | 否 | 0.0 - 1.0 | 控制編碼圖像與潛在表徵的混合強度。值為 1.0 時會完全替換初始影格，較低的值則會進行混合。(預設值: 1.0) |
| `bypass` | BOOLEAN | 否 | - | 繞過條件設定。啟用時，節點將返回未更改的輸入潛在表徵。(預設值: False) |

**注意：** 根據 `latent` 輸入的寬度和高度，`image` 將自動調整大小以符合 `vae` 編碼所需的空間尺寸。
