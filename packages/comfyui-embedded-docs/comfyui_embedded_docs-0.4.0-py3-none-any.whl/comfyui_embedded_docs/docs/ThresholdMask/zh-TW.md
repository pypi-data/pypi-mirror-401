ThresholdMask 節點透過應用閾值將遮罩轉換為二值遮罩。它將輸入遮罩中的每個像素與指定的閾值進行比較，並建立一個新的遮罩，其中高於閾值的像素變為 1（白色），而低於或等於閾值的像素變為 0（黑色）。
