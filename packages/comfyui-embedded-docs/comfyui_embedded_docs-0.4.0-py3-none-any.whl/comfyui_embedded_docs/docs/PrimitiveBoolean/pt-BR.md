O nó Boolean fornece uma maneira simples de passar valores booleanos (verdadeiro/falso) pelo seu fluxo de trabalho. Ele recebe um valor booleano de entrada e emite o mesmo valor inalterado, permitindo que você controle parâmetros booleanos em outros nós.

## Entradas

| Parâmetro | Tipo de Dados | Obrigatório | Intervalo | Descrição |
|-----------|-----------|----------|-------|-------------|
| `value` | BOOLEAN | Sim | true<br>false | O valor booleano a ser passado pelo nó |

## Saídas

| Nome da Saída | Tipo de Dados | Descrição |
|-------------|-----------|-------------|
| `output` | BOOLEAN | O mesmo valor booleano que foi fornecido como entrada |
