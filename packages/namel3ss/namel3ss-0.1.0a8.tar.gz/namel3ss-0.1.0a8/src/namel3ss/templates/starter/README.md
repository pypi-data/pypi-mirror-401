# {{PROJECT_NAME}}

Small starter app with one page and one flow.

Run:
- n3 app.ai check
- n3 app.ai ui
- n3 app.ai actions
- n3 app.ai

Bind the tool:
- n3 tools bind --auto
- n3 app.ai flow "demo"

Export UI contract:
- n3 app.ai ui export

## Secrets setup

This app may require secrets depending on providers/targets. Studio → Setup shows what’s missing.
