Command Line Arguments

-ccs=HIDE               Disable initial display of Common Connection Screen
-connect=CONNECT        Connects to a device; eg -connect=USB::QTL1999-04-013
-devdebug=ON            Enable development debug - strange and fascinating, to some, things may happen
-logconsole			    log to the console as well as to file
-loglevel     		    specifies the logging level [OFF,FATAL,ERROR,WARN,INFO,DEBUG,TRACE,ALL]
-qissamelogging		    forces QIS to use the same logging level as QPS
-logviewer=DIR          Specifies the log viewer file directory
-open=FILE              Opens an archived file; eg -open=C:\>path-to-file\recording.qps
-perfcontrols=ON        Enable performance controls
-port=INT               Allows user to specify port for QPS to use
-qisport=INT            Allows user to specify QIS port to use and listen to
-qisrestport=INT        Allows user to specify QIS rest port to use
-qissamelogging=ON      Inherit logging settings when this instance of QPS launches QIS
-shownotifications=OFF  Disable displaying user message notifications

--help                  Command line help
--version               Version number
