# jira-agile Examples

This directory contains detailed usage examples for the jira-agile skill. Each file focuses on a specific workflow area.

## Available Examples

| File | Description | Key Scripts |
|------|-------------|-------------|
| [epic-management.md](epic-management.md) | Create epics, add issues, track progress | `create_epic.py`, `add_to_epic.py`, `get_epic.py` |
| [sprint-lifecycle.md](sprint-lifecycle.md) | Sprint creation, start, close, issue moves | `create_sprint.py`, `manage_sprint.py`, `move_to_sprint.py`, `get_sprint.py` |
| [backlog-management.md](backlog-management.md) | View and rank backlog issues | `get_backlog.py`, `rank_issue.py` |
| [estimation.md](estimation.md) | Story point estimation and summaries | `estimate_issue.py`, `get_estimates.py` |
