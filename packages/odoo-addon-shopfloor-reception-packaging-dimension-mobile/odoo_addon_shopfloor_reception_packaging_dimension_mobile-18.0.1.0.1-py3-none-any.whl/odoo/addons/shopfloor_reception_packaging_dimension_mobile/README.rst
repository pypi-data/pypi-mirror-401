This module adds the front end part for the
shopfloor_reception_packaging_dimension module. Allowing to set
dimension on packaging related to the product being processed, if they
are not set already. The option needs to be enable on the shopfloor
menu.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_
